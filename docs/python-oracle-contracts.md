# Frozen Python oracle contracts

This record preserves the authority for the first direct-contract tests moved
off the live Python migration oracle. The producing repository baseline was
`7d54261c33a63d248c2ec80dfe7776eab5dc054b`; the selected oracle was CPython
3.13.13 with Unicode database 15.1.0.

The expected values remain inline beside the Node behavior they constrain.
They are not regenerated by Node and must not be replaced with parsed-JSON
equality or a comparison between two Node code paths. Git history contains the
retired implementation. To re-adjudicate a contract, check out the baseline in
a separate worktree and run the recorded command there.

```sh
git worktree add /tmp/iw-python-oracle-contracts \
  7d54261c33a63d248c2ec80dfe7776eab5dc054b
cd /tmp/iw-python-oracle-contracts
PYTHON=python3 npm run build:test
PYTHON=python3 node --test \
  .node-test/node-tests/drift-policy.test.js \
  .node-test/node-tests/exact-plan-apply.test.js \
  .node-test/node-tests/json.test.js \
  .node-test/node-tests/paths.test.js \
  .node-test/node-tests/rest-collector-python-parity.test.js \
  .node-test/node-tests/zscaler-assessment.test.js
```

Use a supported interpreter reported by the baseline resolver if `python3` is
not CPython 3.13 with UCD 15.1.0. This command executes the original live
Python comparisons, including the fake-Terraform Apply fixture, rather than a
reconstructed approximation.
